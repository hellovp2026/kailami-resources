Executive Assistant mock task

Build your EA portfolio.

Three complete mock tasks that prove you can manage inboxes, calendars, and files like a pro. Follow each one, screenshot your work, and you'll have portfolio-ready proof in a weekend.

Tools needed
Gmail, Google Calendar, Google Drive
Time commitment
4–6 hours total
Difficulty
Beginner-friendly
01
Inbox zero mock (Gmail)

If you don't have experience yet, this is one of the best ways to show clients you can manage an inbox. Build a messy inbox, organize it with labels, filters, and templates — then screenshot the transformation.

Watch first
Email Management Training for Beginners — Zero Inbox Tutorial
↗
1
Create a new Gmail account

Create a separate Gmail account you'll use only for practice or mock work. Don't use your personal email — keeps your portfolio clean and professional.

  • mockclient.coach@gmail.com
  • samplebusiness.va@gmail.com
  • practice.executive@gmail.com
2
Make the inbox look realistic

Send emails to yourself or ask a friend to help. Aim for around 30–50 emails so the inbox looks busy and unorganized at first. Create different types:

  • Client inquiries
  • Newsletters or promotions
  • Order confirmations
  • Meeting requests
  • Follow-up emails
  • Spam-like emails
3
Create labels (folders)

Set up labels like Clients, Leads, Urgent, Follow-ups, Newsletters, Finance, and Spam. This shows you know how to prioritize emails, not just read them.

4
Set up filters and rules

Create filters so Gmail automatically organizes emails. Examples:

  • Emails with "invoice" → Finance
  • Emails from certain domains → Clients
  • Newsletters → Newsletters (auto-archive)
  • Emails with "meeting" → Urgent
Why this matters
Filters are a big plus for clients — they prove you can automate repetitive work and save hours every week.
5
Create professional email templates

Save email templates as drafts for:

  • Acknowledging inquiries
  • Confirming meetings
  • Following up on emails
  • Rescheduling requests
  • Out-of-office or availability replies

Keep them polite, clear, and easy to edit. Shows clients you can respond quickly and consistently.

6
Clean and organize the inbox

Now organize everything: archive old emails, mark important ones, flag follow-ups, and aim for zero or near-zero inbox. Take before-and-after screenshots.

Screenshots to capture
  • BEFORE: messy, cluttered inbox with 30+ unread
  • AFTER: organized inbox, zero unread, labels visible in sidebar
  • Filters dashboard showing your automation rules
  • Your saved email templates in drafts
02
Calendar management mock (Google Calendar)

Managing an executive's calendar is one of the most trusted EA skills. This mock shows you can protect their time, avoid conflicts, and keep everything organized.

Watch first
Calendar Management Walkthrough for Virtual Assistants
↗
1
Create a mock calendar

Use your mock Gmail or personal Google account. Create a new calendar named like "Sample Client – Coaching Business" or "Mock Executive Calendar" — keeps it separate from personal and cleaner for portfolio use.

2
Set the client's rules

Before adding meetings, define how the client works:

  • Work days: Monday to Friday
  • Working hours: 10:00 AM – 4:00 PM
  • No meetings on Fridays
  • 15-minute buffer between meetings
  • Maximum of 4 calls per day
Why this matters
This shows clients you know how to protect their time — not just fill their calendar with meetings.
3
Add realistic events

Add events that normally happen in real work:

  • 1:1 client calls
  • Discovery or sales calls
  • Team meetings
  • Admin or focus time
  • Breaks and buffer time

Mix one-time and recurring events for realism.

4
Use color coding

Assign colors so the calendar is scannable at a glance:

  • Client calls — Blue
  • Internal meetings — Green
  • Admin work — Purple
  • Personal / breaks — Yellow
5
Add complete event details

Open each event and include:

  • Meeting link (Zoom or Google Meet)
  • Agenda or purpose of the call
  • Attendees (sample emails)
  • Notes or reminders

Small details like these show clients you're careful and organized — exactly what EAs are hired for.

6
Practice handling changes

In real work, changes are constant. Practice:

  • Rescheduling a meeting
  • Canceling a meeting properly
  • Fixing double bookings
  • Keeping buffer time and rules intact

Take before-and-after screenshots for each scenario.

Screenshots to capture
  • BEFORE: messy calendar with overlapping meetings, no color coding
  • AFTER: clean weekly view with color-coded events
  • AFTER: monthly view showing consistent structure
  • Event detail view with meeting link, agenda, and attendees
  • Before-and-after of a conflict resolution scenario
03
File organization mock (Google Drive)

Shows clients you can organize files and follow a system — a core admin skill that makes you immediately useful on day one.

1
Create a mock Google Drive folder

Use your mock Gmail. Create a main folder named like "Sample Client – Online Business" or "Mock Client – Admin Files". This becomes your "client folder".

2
Create main folders (top-level)

Inside the main folder, create basic folders most clients use:

  • 01_Admin
  • 02_Finance
  • 03_Marketing
  • 04_Clients
  • 05_Operations
  • 06_Resources
  • 07_Archive

Numbering keeps everything organized and consistent.

3
Create subfolders

Inside each main folder, add subfolders:

  • 01_Admin — Contracts, SOPs, Meeting Notes
  • 02_Finance — Invoices, Receipts, Expenses
  • 03_Marketing — Content, Graphics, Captions
  • 04_Clients — Client A, Client B, Client C

This shows clients you know how to structure, not just create folders.

4
Add sample files

Create or upload sample files: Google Docs (meeting notes, SOPs), Google Sheets (trackers, lists), sample PDFs, sample images. Use file names like:

  • 2024-01_ClientA_MeetingNotes
  • Invoice_Jan2024
  • Content_Calendar_Week1
5
Apply a clear naming convention

All files should follow a consistent format:

  • YYYY-MM-DD_FileName_Version
  • ClientName_DocumentType_Date

Shows you're detail-oriented and files are easy to find.

6
Clean and organize

Pretend the drive is messy, then fix it:

  • Move files to correct folders
  • Remove duplicates (or move to Archive)
  • Rename unclear file names
  • Group related files together

Create a messy folder first, then show the organized version — this is what makes the before-and-after powerful.

Screenshots to capture
  • BEFORE: random file names, no folder structure
  • AFTER: clean numbered folders with proper naming
  • Subfolder view showing consistent organization
  • A sample file opened showing the naming convention
